# ScanNHelp

A modern, QR-based security and health management platform.

## Project Structure

- `frontend/`: React + Vite application for the user interface.
- `backend/`: Placeholder for future backend integration.

## Getting Started

### Frontend

1. Navigate to the `frontend` directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
